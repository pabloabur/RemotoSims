Time course of the state during the pulse for the {\itshape activation} states and before and after the pulse for the {\itshape inactivation} states. 

The value of the state $v$ is computed according to the following equation\+:

\begin{equation} v(t) = 1 + (v_0 - 1)\exp[-\alpha(t-t_0)] \end{equation} where $t_0$ is the time at which the pulse changed the value (on to off or off to on) and $v_0$ is value of the state at that time.
